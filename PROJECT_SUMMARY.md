## 🛠️ 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript ES6+
- **构建工具**: Vite
- **图像处理**: Canvas API, Web Workers
- **组件架构**: Web Components (Custom Elements)
- **样式方案**: CSS Custom Properties, BEM 命名
- **路由方案**: 简单 SPA 路由
- **国际化**: i18n 预留架构
- **SEO**: Meta 标签, JSON-LD, sitemap
